# Essential NestJS Scaffold

[Nest](https://github.com/nestjs/nest) is a progressive [Node.js](http://nodejs.org) framework for building efficient and scalable server-side applications.

## Installation

```bash
  npm install
```

## Running the app

```bash
# development
  npm run start

# watch mode
  npm run start:watch

# debug mode
  npm run start:debug
```

## Test

```bash
# unit tests
  npm run test

# e2e tests
  npm run e2e
```
